# Data Model: Todo List Application

**Date**: 2026-01-01
**Feature**: [Todo List Application](../spec.md)

This document defines the data structures for the feature. The primary entity is the `TodoItem`.

## Entity: TodoItem

Represents a single task in the todo list.

### Attributes

| Attribute   | Type      | Description                               | Constraints                             |
|-------------|-----------|-------------------------------------------|-----------------------------------------|
| `_id`       | ObjectId  | Unique identifier for the document.       | Automatically generated by MongoDB.     |
| `title`       | String    | The main title of the task.               | Required, Min length 4, Max length 100. |
| `description` | String    | A more detailed description of the task.  | Optional, Max length 500.               |
| `dueDate`     | Date      | The date when the task is due.            | Required.                               |
| `isDone`      | Boolean   | The completion status of the task.        | Required, Defaults to `false`.          |
| `createdAt`   | Date      | Timestamp of when the task was created.   | Automatically generated.                |
| `updatedAt`   | Date      | Timestamp of the last update.             | Automatically generated.                |

### Mongoose Schema (for Node.js/MongoDB)

This is the proposed Mongoose schema that will be used in the Next.js backend to model and validate the `TodoItem` data in MongoDB.

```javascript
import mongoose from 'mongoose';

const TodoItemSchema = new mongoose.Schema(
  {
    title: {
      type: String,
      required: [true, 'Title is required.'],
      minlength: [4, 'Title must be at least 4 characters long.'],
      maxlength: [100, 'Title cannot be more than 100 characters long.'],
      trim: true,
    },
    description: {
      type: String,
      maxlength: [500, 'Description cannot be more than 500 characters long.'],
      default: '',
    },
    dueDate: {
      type: Date,
      required: [true, 'Due date is required.'],
    },
    isDone: {
      type: Boolean,
      default: false,
    },
  },
  {
    timestamps: true, // Automatically adds createdAt and updatedAt fields
  }
);

export default mongoose.models.TodoItem || mongoose.model('TodoItem', TodoItemSchema);
```
